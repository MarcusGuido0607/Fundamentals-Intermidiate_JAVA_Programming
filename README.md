### 2. Control Flow & Algorithmic Thinking

* **Iteration (Loops):** `LoopForSum`, `LoopWhile`, `LoopDoWhile`
* **Conditional Logic:** `IfIncome`, `LeapYear`, `PalindromeCheck`, `LargestSmallest`

* **Array Traversing & Sorting:** `ArrayTraversing`, `ArrayOddEvenPrime`, `Array1DArithmetic`

### 4. Advanced Concepts: File I/O
Reading, writing, and manipulating external files using Java's built-in file classes (`FileHandling`, `filehand`). Demonstrates persistent data storage concepts outside of active memory.

## 🌟 Final Project: Parking Management System
The highlight of this repository is a fully functional GUI-based Parking Management application located in the `parkingsystem_Final_Project` directory. This project synthesizes all intermediate learnings into a single, cohesive application.

**Key Architecture & OOP Principles Applied:**
* **Inheritance & Polymorphism:** A base `Vehicle` class extended by specific models (`Car`, `Motorcycle`, `Truck`).
* **Encapsulation:** Secure data handling within the `ParkingLot` and `ParkingSystem` classes to manage capacity and ticket logic.
* **Graphical User Interface (GUI):** Event-driven programming implemented via `ParkingSystemGUI` using Java Swing/AWT to create an intuitive user experience.
